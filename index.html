Solar System Model

In this project, I created a 2D animated model of the solar system using Java.
The planets orbit the sun and there are moons which can orbit planets. Although this project was a piece of coursework for my Advanced Programming module, I really enjoyed creating it and added some extra features of my own.
The user is able to increase or decrease the speed of the model at any time through keybinds or pause it altogether. I also added an asteroid belt which consisted of randomly sized and generated asteroids and I also added randomly generated stars too. This project was created in Java but I hope at some point in the future that I can extend on it and make it 3D with correct textures for the planets.




Prison Escape Game

This project was for my Computer Science A Level in college and was programmed in C#.
We had the choice to choose our project idea and so I picked a prison escape game which was inspired by an already existing game called The Escapsits. We had to create the game entirely ourselves including the graphics.
Some key features of the game include: Animated player movements, Camera follows the player, AI for the guards, items such as money and keys, strength which effects attack damage, attacking or bribing guards, and escaping or dying.
Some ways that the guards AI works is using Dijkstra's Shortest Path Algorithm which operates using a computer-generated ajacency matrix to transform the map which has bounding boxes for collision detection into nodes and edges, and the guards also used a finite state machine to easily change the state of the guards. I enjoyed making this project due to the freedom we had but unfortnately it got cut short due to COVID-19.




Graphics Editor

This project was for my Computer Science Group Work module at university and was coded in Java.
The main purpose of this project was to create a pixel-based graphics editor similar to Microsoft Paint. Some of the key features that I worked on was the fill tool using a flood fill algorithm and breadth first search for efficiency, layering functionality so the user can edit on top of other pixels, delete and remove layers using a hash map. I also worked on the select tool functionality, renaming layers, exporting to png including an option to have a transparent background. Finally, I also implemented file saving and loading features so the user can reload the program and open a file and continue editing. I implemented this using serializable in Java and grabbing all the required data to save from the corresponding classes.




Networking Applications

This project was a coursework element for my Computer Networks module and is written in Python.
The goal of this project is to create certain networking tools using pythons Socket library. I was able to implement a ping function to calculate the delay when pinging a certain host. I also created traceroute and paris traceroute functionality to try to accurately get the path that a packet will go through across networks. I implemented paris traceroute specifically for accuracy when dealing with networks that have load balancers and it should remove most of the 'false links' that can sometimes be created with traditional traceroute.
In this project I also implemented a web server and proxy functions. The proxy function specifically handles HTTP get requests and also has caching so that when the client requests the same page it is obtained from the cache just like a typical web proxy does.




Minesweeper Remake

Music Festival Scheduler

In this project, I was tasked to create a scheduler for a music festival.
The scheduler allows for the user to input an act name with their priority, and duration on stage. The user can then set a target time for the highest priority act to be at and they can also set a gap between acts so that they can move equipment etc.
The program also allows for a file upload for festivals with a large amount of acts and also a save to file option for ease of use. The text generated section also allows copying so the user can do that instead if they wish.
